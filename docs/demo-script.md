# Demo Script

## 2-4 Minute Demo Narration

### Opening (30 seconds)

"TownOps Skill is an agent-usable town operations service for NANDA Town. It gives autonomous agents a clean way to report, prioritize, assign, and resolve municipal issues through structured API endpoints and a comprehensive SKILL.md."

### Show the Landing Page (15 seconds)

"The landing page explains the workflow: Report → Triage → Assign → Track. Three CTAs lead to the demo, SKILL.md, and API inspector."

### Demo the Issue Board (45 seconds)

"Here's the issue board showing 10 demo issues across NANDA Town zones. Each issue has a severity badge, status indicator, priority score, and department assignment. We can filter by status, zone, and severity."

[Click "Report Issue" and create a new issue]

"When we report a new issue — say a water leak on Main Street — the service automatically classifies it using AI, assigns a priority score based on weighted factors, and suggests the next action."

### Walk Through an Issue Detail (45 seconds)

[Click into an issue]

"Here's the full issue detail. We see the description, zone, priority score, affected residents, and tags. The Agent Actions panel shows valid status transitions. We can move this issue through the workflow."

[Click "Move to Triaged"]

"The status is validated and updated. The timeline tracks every transition with timestamps and who made the change."

### Generate a Resident Update (30 seconds)

[Click "Generate Resident Update"]

"The AI generates a clear, resident-facing update message. This can be published to community channels. It's grounded in the actual issue state — no hallucination."

### Show the SKILL.md (30 seconds)

[Open SKILL.md viewer]

"The SKILL.md is the most important artifact. It documents every endpoint, every schema, every valid transition. Any autonomous agent can read this and operate the entire workflow without human guidance."

### Show the API Inspector (30 seconds)

[Open API Inspector]

"The API inspector shows all 8 endpoints with their schemas, parameters, and example requests. You can test endpoints directly from here."

### Show Architecture (15 seconds)

"The architecture page explains what's AI-driven, what's deterministic, and why agents can use this autonomously."

### Closing (15 seconds)

"TownOps Skill proves that model-driven agents can operate practical town-service workflows. The SKILL.md alone is sufficient for agent use. The service is live, the endpoints are stable, and the Phase 1 PR path is prepared."
